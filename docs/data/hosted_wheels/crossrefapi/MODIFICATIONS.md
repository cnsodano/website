I am using the crossrefapi python pacakge in my Crossref REST API tutorial series. As part of this and other presentations on the topic, I am using pyodide for interactive coding lessons showing how to use this package. To avoid any issues with installation of the package into pyodide, I renamed the file structure so that the package directory matched the distribution name of the package (that is, instead of crossrefapi/src/crossref/... it became crossrefapi/src/crossrefapi/...) and updated the pyproject.toml accordingly.
